# 🎮 GameCraft Pro Ultimate

## 🌟 גיימקראפט פרו אולטימט - סביבת פיתוח משחקים מהפכנית

**GameCraft Pro Ultimate** היא סביבת פיתוח משחקים מתקדמת שמאפשרת לכל אחד ליצור משחקים מקצועיים ללא ידע תכנות מוקדם. המטרה היא להפוך את יצירת המשחקים לנגישה, מהנה ובטוחה משפטית.

### 🎯 מטרת האפליקציה
להפוך את כל אחד ליוצר משחקים מוצלח - האפליקציה מאפשרת לכל אדם, ללא קשר לרמת הידע הטכני שלו, ליצור, לפרסם ולהרוויח ממשחקים מקצועיים בצורה בטוחה, חוקית ומרווחת.

## 🛠️ התקנה והרצה

### דרישות מערכת
- Node.js 16.0+ 
- npm 8.0+
- דפדפן מודרני (Chrome, Firefox, Safari, Edge)

### התקנה מהירה
```bash
# שכפול הפרויקט
git clone <repository-url>
cd gamecraft-pro-ultimate

# התקנת תלויות
npm run install-all

# הרצה במצב פיתוח
npm run dev

# הרצה בפרודקשן
npm run build
npm start
```

### מבנה הפרויקט
```
gamecraft-pro-ultimate/
├── client/          # אפליקציית React (Frontend)
├── server/          # שרת Node.js/Express (Backend)
├── shared/          # קוד משותף
├── docs/           # תיעוד
└── assets/         # נכסים (תמונות, סאונד, וכו')
```
