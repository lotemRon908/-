### 🚀 Getting Started

1. **Clone and Install**
   ```bash
   git clone <repository>
   cd gamecraft-pro-ultimate
   npm run install-all
   ```

2. **Setup Environment**
   ```bash
   cp server/.env.example server/.env
   # Edit .env with your configuration
   ```

3. **Run Development**
   ```bash
   npm run dev
   ```

4. **Access Admin Panel**
   - Navigate to `/admin`
   - Use code: `lotemronkaplan21`
